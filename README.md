# Flask 待办事项应用

一个基于 Python (Flask) 和原生 JavaScript 构建的简单待办事项应用。

## 前置要求

- Python 3.x
- pip

## 安装

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行应用程序：
   ```bash
   python app.py
   ```

2. 打开浏览器并访问 `http://localhost:5000`。

## 功能特性

- 添加待办事项
- 标记待办事项为已完成
- 删除待办事项
- 简洁清爽的用户界面

## 项目结构

- `app.py`: Flask 后端应用程序。
- `templates/index.html`: 主 HTML 模板。
- `static/style.css`: CSS 样式表。
- `static/script.js`: 前端 JavaScript 逻辑。
